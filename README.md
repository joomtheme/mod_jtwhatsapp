# JT WhatsApp Button

JT WhatsApp Button is a simple floating WhatsApp button module for Joomla.

## Features

- Floating WhatsApp contact button
- Custom phone number
- Custom default message
- Left / right position
- English and Turkish language support
- Joomla update support ready

## Requirements

- Joomla 6
- PHP 8.1+

## Installation

1. Download the latest release package.
2. Log in to Joomla Administrator.
3. Go to System -> Install -> Extensions.
4. Upload the module ZIP package.
5. Publish the module and assign it to a template position.
